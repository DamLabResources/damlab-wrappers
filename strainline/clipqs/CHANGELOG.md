# Changelog

All notable changes to the ClipQS wrapper will be documented in this file.

## [1.0.0] - 2025-04-10

- First stable release

- Input validation
- Comprehensive error handling
- Documentation
- Type hints

- Enhanced code structure and organization

- More robust sequence processing
- Improved logging
